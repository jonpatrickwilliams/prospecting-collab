This console saves everything to a Google Sheet you control, so it works whether you open it in Claude, download it, or send it to Ali. This is a one time setup per browser.

First time only:
  1. Open the data sheet: JP-Ali Prospecting Collab - Data Store ↗
  2. Extensions → Apps Script
  3. Delete any placeholder code, paste in the script provided
  4. Deploy → New deployment → type Web app → Execute as Me → Who has access Anyone → Deploy
  5. Authorize when prompted, then copy the Web app URL
  6. Paste that URL below

Already have the URL from a teammate? Just paste it in, no need to redo the sheet setup.
